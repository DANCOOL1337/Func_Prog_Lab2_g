## Задача 1

### Текст задачи

На основе списка, содержащего двоичные значения цифр от 1 до 9, получить их десятичные представления.

### Алгоритм решения

1)Создается список binaryNumbers, который содержит списки двоичных цифр для чисел от 1 до 9

2)Преобразование двоичного списка в десятичное число (binaryToDecimal):
       -Переворачиваем список чтобы идти от младших разрядов
       -Для каждого элемента проверяем значение:
              --Если цифра 1 → добавляем 2^позиция к результату
              --Если цифра 0 → ничего не добавляем
       -Возвращаем итоговое десятичное число

3)Обход списка (processList):
       -Проходим по каждому элементу списка binaryNumbers
       -Для каждого списка цифр вызываем binaryToDecimal
       -Выводим первоначальный двоичный список и полученное десятичное число

4)Выводим на экран все двоичные цифры и их десятичный вид

## Задача 2

### Текст задачи

Список содержит символы, обозначающие шестнадцатеричные цифры. Получить шестнадцатеричное число из этих цифр.

### Алгоритм решения

1)Подготовка данных: Создаем список hexNumbers, который содержит списки символов для шестнадцатеричных чисел (от 1 до FF)

2)Преобразование символа в число (hexCharToValue):
       -Проверяем входной символ
       -Если символ от 0 до 9,то возвращаем число от 0 до 9
       -Если символ от A до F или a до f,то возвращаем число от 10 до 15
       -Иначе ошибка

3)Преобразование списка в десятичное число (hexListToDecimal):
       -Переворачиваем список (List.rev) чтобы идти от младших разрядов
       -Для каждого символа получаем его числовое значение через hexCharToValue
       -Умножаем значение на 16^позиция и добавляем к результату
       -Возвращаем итоговое десятичное число

4)Обход списка (processList):
       -Проходим по каждому элементу списка hexNumbers
       -Для каждого списка символов вызываем hexListToDecimal
       -Выводим исходный список символов и полученное десятичное число
5)Выводим на экран все шестнадцатеричные числа и их десятичный вид
